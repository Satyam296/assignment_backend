# Backend (assignment)

This repository contains the Backend for the assignment.

- Frontend: https://vermillion-strudel-6bb21d.netlify.app/

Usage
1. Copy `.env.example` to `.env` and fill in real values (do NOT commit `.env`).
2. Install dependencies: `npm install`.
3. Run: `npm run dev` (or the script defined in `package.json`).

Notes
- This repo intentionally tracks the frontend URL via `.env.example` and `README.md` so secrets remain out of version control.
